# (Hackintosh)神舟优雅X3G1 EFI
## **After using this EFI to install the system, please use OCLP-mod to patch the Intel network and graphics.**

![](/pic/5a58ba6febe78cf6888f791cc71438a8.jpg)

|          |          | work? |
| -------- | -------- | ----- |
| CPU      | i3-5005u | yes   |
| Graphics | hd5500   | yes   |
| Touchpad | SYN1B7B  | yes   |
| Keyboard | PS2      | yes   |
| Camera   |          | yes   |
| Audio    |          | yes   |
| sleep    |          | no    |

Sleep mode can cause the PS2 keyboard to stop working, so if you often close the lid, please disable it.
This EFI supports macOS 15. If using it causes any abnormalities on your computer, I am not responsible.
